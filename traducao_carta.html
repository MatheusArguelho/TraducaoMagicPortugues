Texto em inglês:

Flying, lifelink
Ward—Discard a card.
Whenever a modified creature you control attacks, double the number of each kind of counter on it. Then for each nontoken permanent attached to it, create a token that's a copy of that permanent attached to that creature.

Tradução:

Voar, vínculo vitalício
Ward – Descarte uma carta.
Toda vez que uma criatura modificada que você controla atacar, dobre o número de cada tipo de marcador nela. Então, para cada permanente que não seja uma ficha anexada a ela, crie uma ficha que seja uma cópia daquela permanente anexada àquela criatura.
